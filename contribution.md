# Contributing to Blackcrow Escrow

Thanks for your interest in contributing! 🚀

## 🎯 Project Goal

Build a reliable escrow system to eliminate fraud in social commerce.

---

## 🧩 Areas You Can Contribute To

### Frontend (Expanded)

- Improve UI responsiveness (mobile fixes needed)
- Enhance dashboard UX (`user.html`)
- Add animations and feedback states
- Polish landing page (`landingpage.html`) and auth pages (`login.html`, `signup.html`)
- Improve accessibility (ARIA labels, keyboard navigation)
- Add form validation and error handling

### Backend (Needed)

- Build REST API (Node.js / Flask)
- Database design (transactions, users, escrow states)
- Payment integration

### Features

- Full authentication system (integrate with backend)
- Notifications (email/SMS)
- Dispute resolution flow
- Escrow release logic automation
- Multi-currency support

---

## 🏁 Getting Started

1. Fork the repo
2. Clone it
3. Create a branch
4. Make your changes
5. Submit a PR

---

## 📏 Guidelines

- Keep code clean and readable
- Follow existing structure (dark theme, Manrope font)
- Comment complex logic
- Test your changes before submitting (run local server for favicon/assets)
- Ensure responsive design on mobile
